HTML

  • Html elements provide content and basic framework for all pages on the web.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Declarations are not case sensitive but HTML elements are case sensitive.
  • The title element defines title for webpage and is extremely important for Search Engine Optimization.
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage.
  • The link element connects an exteral resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The script element embeds client side scripts in the HTML documents.
  • The child element is nested inside other element called Parent element.
  • The div element creates division in page.
  • The span element creates inline division.
  • Element refers to the entire element from opening tag to closing tag.
  • Tag refers to only what appears inside the angle brackets.
  • Biggest heading should only appear once on each page.
  • p element creates paragraphs.
  • ul element creates unordered list which are bulletpoints.
  • ol element creates ordered list which are numbered.
  • li stands for list items
  • img element contains information about images that are displayed on the webpage and can contain various attributes.
  • src attribute defines the location of the image.
  • alt attribute containes text string that describes the appearance functionality of the image.
  • br creates a line of empty space.
  • a element stand for anchor and creates links to same webpage or other webpages/
  • href attribute points to the URL for link
  • footer element appears at the bottom of the page and contain information such as author, copyright, contact, sitemap, and navigation.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Selector defines the element or attributes to which the rules or declarations will apply.
  • Declarations contains two important components which is the CSS property we want to apply and value of the property
  • We can combine two selectors or more into a single CSS rule with a comma.
  • Combined selectors will share same styling.
  • Link to our style sheet will be entered in the head element.
  • Link element creates the connection to an external file.
  • rel attributes specifies the relationship between the current document and the linked document resource.
  • Class attributes allow us to share CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone "url": retrieves repo from hosted location via url.
  • git add -A: adds new branch to current working branch.
  • git commit -m "brief description of change": commit repo and message that describes what the commit contains.
  • git checkout main: switches you to main branch.
  • git pull origin main: pulls down latest main branch.
  • git push origin "with file name without the quotes": pushes up all our changes

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • To declare a variable use the var keyword.
  • After your var keyword, we give the variable a name, which must be unique and reference info stored in variable.
  • Use an assignment operator = to assign the sepecific data that we want stored in the variable in our code.
  • Ex: var shape="circle";
  • Variable contains string and is contained within the quotation marks.
  • Each line of JS ends with a semicolon ;.
  • console.log method outputs a message to the web console by adding an arguement and makes it easier to code.
  • Connect script.js to index.html by adding script element under the footer
  • Control flow is the order in which a computer executes code in a script.
  • Codes run top to bottom.
  • Control flow can be interrupted if we use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • if statement is a way of introducing decision-making into our code and will only execute if the statement is true, or meets a condition.
  • We use else if statement if there is more than one condition.
  • === is a strict equality operator and checks to see if two values are equal.
  • Array is a single variable that is used to hold a group of data which makes it useful to perform task on multiple pieces of data at once.
  • Array index starts at 0.
  • for loop uses predictable pattern of indices to perform task on all items in an array by allowing a single code block to be executed over and over.
  • In order to use for loop we need index and array length.
  • Array length is important because the for loop needs to know how many times the code will run until there are no more items left in the array.
  • Length can be found using a built in property of an array, named length.
  • For loop contains 3 important statements:
    Starting point for our loop.
    Condition
    Code block; what allows the array to iterate over each item.
